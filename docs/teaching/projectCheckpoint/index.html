By end of Week 5, your video game should include the following features:

  1. Individual Actor Sprites
    the game should use your own choice of pictures
  2. Collectibles
    there should be an item the player collects (e.g gems in gemcatcher)
  3. Collision events
    what happens when your player crashes into walls, collectibles and other characters
  4. Active scoreboard
    show the current score on screen and update in real time
  5. Internal comments
    Use comments to explain what your code is doing.
